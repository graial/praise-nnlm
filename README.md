
## Env setup

 - Install direnv to make your life easier :) 
`https://direnv.net/`

 - clone the repo
With python installed:
`python -m venv .`
In case your os uses python2 by default
`python3 -m venv .`

 - Install required python libraries (using Tensorflow on CPU).  You're on your own if you wanna setup via GPU: 
`pip install -r environment.txt`
